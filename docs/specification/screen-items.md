# 画面項目一覧

## 共通仕様

- 必須項目には「必須」を表示し、エラーは項目の近くに理由と修正方法を表示する。
- 日付は利用者のタイムゾーンで `YYYY/MM/DD` 表示、DB では ISO 形式で保持する。
- 一覧は 20 件単位を初期値とし、0 件時には次の操作を案内する。
- 保存・削除・PDF 出力の結果は、画面上の通知と読み上げ可能な文言で伝える。
- 削除は確認ダイアログを表示し、復元可能な期間を案内する。

## 認証・ホーム

### SCR-001 ログイン

| 種別 | 項目・操作     | 必須 | 仕様                                                     |
| ---- | -------------- | ---- | -------------------------------------------------------- |
| 入力 | メールアドレス | ○    | メール形式、254 文字以内                                 |
| 入力 | パスワード     | ○    | 8 文字以上、表示切替あり                                 |
| 操作 | ログイン       | -    | 成功時は SCR-010、失敗時は認証情報を特定しない共通エラー |
| 操作 | ユーザー登録   | -    | SCR-002 へ遷移                                           |

### SCR-002 ユーザー登録

| 種別 | 項目・操作             | 必須 | 仕様                                    |
| ---- | ---------------------- | ---- | --------------------------------------- |
| 入力 | 氏名                   | ○    | 100 文字以内                            |
| 入力 | メールアドレス         | ○    | 一意、メール形式                        |
| 入力 | パスワード             | ○    | 8 文字以上、確認入力と一致              |
| 入力 | 現在の職種・役職・所属 | -    | 各 100 文字以内                         |
| 操作 | 登録                   | -    | 利用者と既定設定を作成し SCR-010 へ遷移 |

### SCR-010 ダッシュボード

| 領域   | 表示項目・操作                                                         | 仕様                                                               |
| ------ | ---------------------------------------------------------------------- | ------------------------------------------------------------------ |
| KPI    | 総合達成率、進行中、達成済み、期限超過、今週・今月の活動時間、継続日数 | 対象は未削除データ。総合達成率は目標の評価上の重みで加重平均する。 |
| グラフ | 目標別達成率、月別活動時間                                             | 値と凡例を表示し、グラフ以外の一覧でも確認可能にする。             |
| 要対応 | 期限が近い目標、遅延目標、今週実施すべき項目                           | 期限、状態、目標詳細へのリンクを表示する。                         |
| 活動   | 最近の活動、日次実績追加                                               | SCR-060 を 1 操作で開始できる。                                    |

## キャリア・スキル・ロードマップ

### SCR-020 キャリア目標一覧

| 種別 | 項目・操作                                                     | 仕様                          |
| ---- | -------------------------------------------------------------- | ----------------------------- |
| 絞込 | ステータス、優先度、期限                                       | 複数条件を併用できる。        |
| 一覧 | 目標名、目指す役割、期限、優先度、ステータス、ロードマップ進捗 | 行選択で SCR-022 へ遷移する。 |
| 操作 | 新規作成、編集、アーカイブ                                     | 新規・編集は SCR-021。        |

### SCR-021 キャリア目標編集

| 項目                               | 必須 | 入力規則                             |
| ---------------------------------- | ---- | ------------------------------------ |
| キャリア目標名                     | ○    | 1～200 文字                          |
| 目指す役割                         | ○    | 1～200 文字                          |
| 目標期限                           | ○    | 登録日以降                           |
| 設定理由、現在の状態、達成時の状態 | -    | 各 2,000 文字以内                    |
| 現在の役割                         | -    | 200 文字以内                         |
| 優先度                             | ○    | 高・中・低                           |
| ステータス                         | ○    | 未着手・進行中・達成・保留・中止     |
| 関連スキル                         | -    | SCR-030 に登録済みのスキルを複数選択 |

### SCR-022 キャリア目標詳細

| 領域         | 表示項目・操作                               |
| ------------ | -------------------------------------------- |
| 概要         | SCR-021 の登録内容、編集、アーカイブ         |
| スキル差分   | スキル名、現在レベル、目標レベル、差分       |
| ロードマップ | 項目名、期間、状態、進捗、SCR-040 へのリンク |

### SCR-030 / SCR-031 スキル一覧・編集

| 種別 | 項目・操作                               | 必須・仕様                               |
| ---- | ---------------------------------------- | ---------------------------------------- |
| 絞込 | 分類、レベル、名称                       | 名称は部分一致                           |
| 一覧 | 名称、分類、現在レベル、目標レベル、差分 | 1～5 を表示                              |
| 入力 | スキル名                                 | 必須、同一ユーザー内で一意、100 文字以内 |
| 入力 | 分類                                     | 必須、設定済み分類から選択               |
| 入力 | 現在レベル、目標レベル                   | 必須、1～5                               |
| 入力 | 判定基準、備考                           | 任意、各 2,000 文字以内                  |

### SCR-040 / SCR-041 ロードマップ・項目編集

| 種別     | 項目・操作                                                             | 必須・仕様                                        |
| -------- | ---------------------------------------------------------------------- | ------------------------------------------------- |
| 表示切替 | 時系列、一覧                                                           | MVP は 2 形式。スマートフォンは一覧を既定にする。 |
| 一覧     | 項目名、キャリア目標、対象スキル、開始・完了予定日、優先度、状態、進捗 | 期限順または表示順で並べる。                      |
| 入力     | 項目名                                                                 | 必須、200 文字以内                                |
| 入力     | キャリア目標                                                           | 必須、所有ユーザーの目標から選択                  |
| 入力     | 対象スキル                                                             | 任意、所有ユーザーのスキルから選択                |
| 入力     | 開始・完了予定日                                                       | 必須、開始日≦完了日                               |
| 入力     | 前提項目                                                               | 任意、同一キャリア目標内。循環参照は禁止          |
| 入力     | 優先度、ステータス                                                     | 必須                                              |
| 操作     | 目標を作成                                                             | 選択項目を初期値にして SCR-051 へ遷移             |

## 目標・日次実績

### SCR-050 目標一覧

| 種別 | 項目・操作                                              | 仕様                                 |
| ---- | ------------------------------------------------------- | ------------------------------------ |
| 絞込 | ステータス、分類、評価期間、期限、遅延有無              | 条件を解除できる。                   |
| 一覧 | 目標名、分類、期限、現在値/目標値、達成率、予定差、状態 | 文字とアイコンで遅延状態を表示する。 |
| 操作 | 新規作成、詳細、編集、アーカイブ                        | 詳細は SCR-052。                     |

### SCR-051 目標編集

| 項目                             | 必須     | 入力規則                                                   |
| -------------------------------- | -------- | ---------------------------------------------------------- |
| 目標名、説明、分類               | ○        | 名称 200 文字、説明 2,000 文字以内                         |
| ロードマップ項目                 | -        | 選択時は所有関係を検証する。                               |
| 評価期間                         | -        | 登録済み期間から選択                                       |
| 開始日、期限                     | ○        | 開始日≦期限                                                |
| 達成条件、測定方法               | ○        | 各 2,000 文字以内                                          |
| 計算方式                         | ○        | 数値・マイルストーン・習慣・手動                           |
| 目標値、現在値、単位             | 条件付き | 数値型で必須。目標値 > 0                                   |
| 計画日数                         | 条件付き | 習慣型で必須。1 以上                                       |
| 手動進捗率、判断理由             | 条件付き | 手動型で必須。進捗率 0～100                                |
| 優先度、評価上の重み、ステータス | ○        | 重みは 0 より大きい数値                                    |
| SMART 判定                       | -        | 5 観点を警告表示する。警告があっても理由を確認して保存可能 |

### SCR-052 / SCR-053 目標詳細・マイルストーン編集

| 領域           | 項目・操作                                   | 仕様                         |
| -------------- | -------------------------------------------- | ---------------------------- |
| 概要           | 達成率、超過率、予定差、期限、状態、達成条件 | 計算根拠を表示する。         |
| マイルストーン | 名称、期限、完了条件、重み、状態、完了日     | 名称と期限と正の重みが必須。 |
| 日次実績       | 日付、内容、時間、進捗量、学び               | SCR-060 へ遷移できる。       |
| 証跡           | 名称、URL、説明、作成日、成果への影響        | URL は http/https。          |

### SCR-060 日次実績入力

| 項目                                         | 必須                     | 入力規則                                       |
| -------------------------------------------- | ------------------------ | ---------------------------------------------- |
| 実施日、対象目標                             | ○                        | 未来日は不可。最近使用した目標を上位表示       |
| 対象マイルストーン                           | -                        | 選択目標に属する項目のみ                       |
| 実施内容                                     | ○                        | 1～4,000 文字                                  |
| 作業時間                                     | ○                        | 分単位、0～1,440                               |
| 進捗量                                       | -                        | 0 以上。数値型目標では現在値への加算候補を表示 |
| 完了したこと、学んだこと、困りごと、次の行動 | -                        | 各 2,000 文字以内                              |
| 自己評価                                     | -                        | 1～5 と任意コメント                            |
| 証跡 URL                                     | -                        | http/https、複数登録可能                       |
| 操作                                         | 自動保存、登録、前日複製 | 自動保存状態と最終保存時刻を表示する。         |

### SCR-061 活動カレンダー

| 領域       | 表示項目・操作                                 |
| ---------- | ---------------------------------------------- |
| カレンダー | 日別の入力有無、活動時間、実績件数             |

## 評価資料・設定

### SCR-070 評価期間一覧

| 項目・操作 | 仕様                                         |
| ---------- | -------------------------------------------- |
| 一覧       | 期間名、開始日、終了日、重点テーマ、資料状態 |
| 登録・編集 | 期間名、開始日、終了日が必須。開始日≦終了日  |
| 資料作成   | SCR-071 へ遷移する。                         |

### SCR-071 評価資料作成

| 領域     | 項目・操作                                                       | 仕様                                   |
| -------- | ---------------------------------------------------------------- | -------------------------------------- |
| 条件     | 評価期間、現在の役割、期待役割、重点テーマ                       | 期間は必須。                           |
| 自動集約 | 目標、達成率、日次実績、証跡、活動時間                           | 対象期間と所有ユーザーで絞る。         |
| 編集     | 定量成果、定性成果、貢献、未達、課題、改善策、次期目標、相談事項 | 自動保存する。                         |
| 操作     | 再集約、下書き保存、プレビュー                                   | 再集約前に手動編集への影響を確認する。 |

### SCR-072 評価資料プレビュー

| 領域       | 表示項目・操作                                     |
| ---------- | -------------------------------------------------- |
| 基本情報   | 期間、役割、重点テーマ                             |
| 目標別結果 | 達成条件、達成率、実績、成果、証跡、未達、振り返り |
| 総括       | 定量・定性成果、貢献、課題、次期目標               |
| 操作       | 編集へ戻る、印刷、PDF 出力                         |

### SCR-080 設定

| 領域       | 項目・操作               | 仕様                                       |
| ---------- | ------------------------ | ------------------------------------------ |
| スキル分類 | 名称、表示順、有効/無効  | 使用中分類は削除せず無効化する。           |
| レベル定義 | レベル 1～5 の名称・説明 | レベル数は MVP では固定する。              |
| アカウント | 氏名、職種、役職、所属   | メール変更とパスワード変更は別確認を行う。 |
